강좌 소개

HTML 의 기본을 차근차근 이해하기 쉽게 설명해 줍니다. 또한 HTML5 에서 새롭게 추가된 부분들을 소개합니다. HTML 은 웹언어중 가장 기본적이고, 쉽고, 많이 사용하고 있는 언어입니다. 블로그나 커뮤니티에 글을 쓸때 보이는 img src ... 이런 것들은 모두 HTML 언어로 작성된 것입니다. 여러분이 보고 계시는 웹페이지는 거의 대부분 HTML 로 만들어 졌지요. 그만큼 널리 쓰이고, 중요한 언어입니다.

운영자 추천

생활코딩의 수업답게 단지 HTML 언어뿐 아니라 웹개발의 기본적인 이야기를 재미있고 알기 쉽게 설명해 줍니다. 특히 meta, seo(검색엔진최적화) 부분은 누구나 쓰지만 자세히는 알지 못하기 쉬운데, 자세하면서도 간단하게, 깊게 이야기 해 줍니다. HTML에 대해 알고 있다고 하더라도 충분히 들을만한 가치가 있는 좋은 강좌입니다.
